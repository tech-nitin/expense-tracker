# 💰 Expense Tracker Dashboard

A modern and responsive Expense Tracker Dashboard built using HTML, CSS and JavaScript.

The application helps users track their income, expenses and spending categories through an interactive dashboard UI.

## 🚀 Features

- 📊 Dashboard overview
- 💵 Monthly Income card
- 💸 Monthly Expenses card
- 💳 Wallet/Card section
- 📈 Expense analytics section
- 🗂 Expense category breakdown
- ➕ Add Income functionality
- ➖ Add Expense functionality
- 📝 Recent Transactions table
- 🗑 Delete transactions
- 📤 Export transactions as JSON
- 💾 Local Storage support
- 📱 Responsive design

---

## ⚙️ Installation

### Clone repository

```bash

```

### Open project folder

```bash
cd expense-tracker
```

### Run project

Simply open:

```text
index.html
```

or use VS Code Live Server.

## 💾 Data Storage

The application uses browser Local Storage to save transactions.

Stored data:

- Income
- Expenses
- Transactions

Data remains available even after refreshing the page.
